## 关于我

浙江大学本科在读，AI 产品经理，ENFJ。

有两段大厂 AI 产品实习经历，做过 **LLM Agent** 和 **C 端用户增长**——接触过声纹识别、AR 眼镜、支付场景里的 AI，也写过比价 agent 的 prompt 和 AI 批改的对话风格。不是泛泛的「AI PM」，是在具体的产品里摸过具体的用户。

我做东西的起点通常是一个人的处境：学生做完作业没人讲解、青少年想找一个私密的成长出口、想学舞蹈但没有教练陪。技术是工具，那个人才是起点。

在 AI 越来越强的时代，我想做那个一直在追问「用户到底感受到了什么」的人——保持对人的温度，做让人觉得「被理解」的产品。

## 产品理念

好产品不只是算法更好一点。

是有人注意到了那些用户懒得开口说的小摩擦，观察到他们围绕一个残缺工具默默养成的习惯，抓住了产品赢得或失去信任的那一瞬间。

我想做那个离这些时刻最近的人——把人的复杂和混乱，翻译成团队可以真正落地的东西。让 AI 越来越强的同时，我们做出来的产品，依然像是为人做的，而不只是对着人发射的。
